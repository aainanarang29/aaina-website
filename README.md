# Aaina Narang - Personal Website

A simple, modern personal website showcasing professional work and personal brand.

## Features

- Clean, minimal design
- Responsive layout (mobile, tablet, desktop)
- Smooth scrolling navigation
- Fade-in animations on scroll
- Professional yet authentic tone

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Smooth scrolling and scroll animations
- `README.md` - This file

## Setup

1. Upload all files to your web hosting service
2. Point your domain (aainanarang.com) to your hosting
3. Ensure `index.html` is in the root directory

## Customization

- Update email and social links in `index.html` (contact section)
- Modify colors in `styles.css` (CSS variables at the top)
- Adjust content in `index.html` as needed

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge).
